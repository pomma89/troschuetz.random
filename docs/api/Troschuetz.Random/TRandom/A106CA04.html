TRandom.

LogisticSamples(double, double) Method

Summary

Returns an infinite sequence of logistic distributed floating point random numbers.
Namespace
Troschuetz.Random
Containing Type
TRandom

Syntax

public IEnumerable<double> LogisticSamples(double mu, double sigma)

Remarks

This method simply wraps a call to Sample, with a prior check of the distribution parameters. If you absolutely need the best performance, you may consider using an instance of LogisticDistribution for each group of parameters.

Parameters

Name Type Description
mu double The parameter mu which is used for generation of logistic distributed random numbers.
sigma double The parameter sigma which is used for generation of logistic distributed random numbers.
