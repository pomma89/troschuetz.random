[Serializable]
public sealed class LogisticDistribution : AbstractDistribution, IContinuousDistribution, 
    IDistribution, IMuDistribution<double>, ISigmaDistribution<double>

Remarks

The implementation of the LogisticDistribution type bases upon information presented on Wikipedia - Logistic Distribution and the implementation in the Boost Random Number Library. The thread safety of this class depends on the one of the underlying generator.
