NR3Q2Generator Class

Summary

A generator whose original code has been found in a famous book about numerical analysis. Inside the book, it is a "backup" generator which should be used NR3Q1Generator has too short a period and NR3Generator is too slow.

Remarks

This generator has a period of ~ 8.5 * 10^37. This generator is NOT thread safe.
